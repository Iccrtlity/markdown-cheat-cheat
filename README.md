## Basics

### Headings
```md
# H1
## H2
### H3
```

### Text
```md
*italic*
**bold**
~~strikethrough~~
```

### Lists
```md
- Item 1
- Item 2

1. First
2. Second
```

### Links & Images
```md
[Google](https://google.com)

![Alt text](image.png)
```

### Inline Code
```md
`code`
```

## README Template (Copy & Paste)

### Minimal

```md
# Project Name

## Description
Short description.

## Installation
Steps here.

## Usage
Example.

## License
MIT
```

---

### Standard Project

```md
# Project Name

## Description
Explain what the project does.

## Features
- Fast
- Simple
- Lightweight

## Installation
```bash
npm install your-package
```

## Usage
```js
import x from "your-package";
```

## Configuration
Explain options.

## Contributing
Guidelines.

## License
MIT
```
