# README Templates

## Minimal README (fast projects)

```md
# Project Name

## Description
What this project does.

## Installation
```bash
npm install
```

## Usage
How to run it.

## License
MIT
```

## Standard README (normal projects)

```md
# Project Name

## Description
Short explanation of the project.

## Features
- Feature 1
- Feature 2

## Installation
```bash
git clone <repo>
cd project
npm install
```

## Usage
```bash
npm start
```

## License
MIT
```
